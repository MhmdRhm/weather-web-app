## Prerequisites 
In order to install and set things up you'll need to have:
1. Mongodb Daemon
2. Mongodb Tools
and optionally
3. Mongodb Compass

## Installation
1. git clone https://github.com/mhmdrhm/weather-web-app.git
2. cd weather-web-app
3. npm install (to install all project's dependencies)
4. open a terminal
5. mongorestore ./dump
